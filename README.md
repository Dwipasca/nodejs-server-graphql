# nodejs-server-graphql

## Depp or Lib I use
- express
- nodemon
- express-graphql
- graphql
- cors

## How to use
- clone this repo
```
  git clone git@github.com:Dwipasca/nodejs-server-graphql.git
```
- go to the project directory
```
  cd nodejs-server-graphql
```
- install depp 
```
  npm i | npm install
```
- run project
```
  npm run dev
```
- open in your browser with url http://localhost:4000/graphql
